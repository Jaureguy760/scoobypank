# ScoobyPanK — CSHL poster supplement

Supplementary material for the CSHL "AI in Biology" Virtual Poster Gallery poster:

**Single-cell sequence-to-function modeling of pancreas and immune cells for large-scale
variant effect prediction in type 1 diabetes**
Jeff Jaureguy, Weston Ellison, Kyle Gaulton, Graham McVicker (Salk Institute · UC San Diego).

This site is the QR-linked supplement: full numbered references, a high-level methods
summary, the reference atlas + benchmarking datasets, and code links.

- `index.html` — the supplement page (published via GitHub Pages).
- `references.json` — structured citation + dataset registry (DOIs verified via CrossRef).
- `scoobypank-poster.pdf` — the poster.

Contact: jjauregu@ucsd.edu
